# 🌾 PadiDoctor AI - Pak Mat (Your Agriculture Expert)

**PadiDoctor AI** is an intelligent AI-powered chatbot specifically designed to assist Malaysian paddy farmers in diagnosing crop diseases, managing fertilizers, and boosting yields via mobile devices, all delivered through the friendly "Pak Mat" persona.

---

## ✨ Key Features
- **Paddy Disease Diagnosis:** Pak Mat identifies paddy diseases (e.g., Leaf Blast, Brown Planthopper) via images.
- **Technical Advice:** Specific fertilizer and pesticide recommendations for the Malaysian market.
- **Friendly Persona:** Relaxed interaction in Malay that resonates with local farmers.
- **Latest Technology:** Powered by the **Gemini 3 Flash** model for lightning-fast responses.

## 🛠️ Tech Stack
- **Frontend:** React.js + Vite
- **Backend:** Node.js + Express
- **AI Model:** Google Gemini 3 Flash
- **Database/Auth:** Firebase
- **Deployment:** Google Cloud Run

## 🚀 Local Setup
1. Install dependencies: `npm install`
2. Configure `.env` (`GEMINI_API_KEY`)
3. Run Backend: `node server.js`
4. Run Frontend: `npm run dev`
